### if и  else
Синатаксис :

    if (<condition>) {<do somthing>} else {<do another thing>}

Здесь condition  логический вектор длины 1.
#### Пример 1:
    if (sqrt(2) > 1.5)
    {
        print("Good")
    }
    else
    {
        print("Bad")
    }
